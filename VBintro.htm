The New Project Window
 

 
 
The New Project window displays every time you start Visual Basic,

or

whenever you select New Project from the File menu.

 

 

 

 

 

 

 
The Toolbar
 

 

 
The Toolbar that you see just below the menu bar changes as you
use Visual Basic. Visual Basic has a total of four toolbars:

Debug, Edit, Form Editor, and Standard.

You can display and hide these toolbars from the Toolbars option
in the View menu. Each toolbar shows multiple buttons that offer
one-button access to commonly used options instead of having to
navigate through menus to find them.

 

 

 

 

 

 

 
The Toolbox
 

 
 
The Toolbox window, typically called the Toolbox, differs from the toolbar.

This is a collection of tools that acts as a repository of controls that you can place on any form you create. A form can hold many controls.

You can add and remove tools from the Toolbox.
  
 

 

 

 

 

 

 
The Project Window
 

 
A Project can consist of several files, so the Project Window enables you to manage all those components and bring the component you want to work on into the editing area.

 
It is also known as the Project Explorer because of its Windows Explorer style interface that lets you expand and shrink object groups.

 

 

 

 

 

 

 
The Form Window
 

 

 
Most of the work done in VB goes on in the Form Window.

 
You'll design all your application's forms, which are the background windows that your users see, in the control editing area (the form) within the form window.

 

 

 

 

 

 

 
The Form Layout Window
 

 
The Form Layout Window is an interesting little window connected closely to the Form Window, because the Form Layout Window shows you a preview of the Form Window's location on the screen.

 
If one or more forms appear in your Form Window, thumbnail sketches of those forms will also appear in the Form Layout Window, showing you where each of the forms will appear on the screen.

 

 

 

 

 

 
The Properties Window
 

 
With a control selected, the Properties Window changes to list every property related to that control.

 
Visual Basic sets each control's initial property values, and you can modify these property values as needed.

 
To assign a value to a property, simply select the property and type a new value. Don't forget to preface each object name with the correct three letter prefix.

 

 

 

 

 

 

 
The Code Window
 

 

 
The Code Window shows all code associated with controls in your form.
